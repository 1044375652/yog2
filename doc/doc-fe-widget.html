widget
  • 功能：调用模板组件，渲染输出模板片段。
  • 使用： {%widget $id%}
  • 用法：在模板中调用某个模板组件
    {%html framework="home:static/lib/mod.js"%}
  {%head%}
     <meta charset="utf-8"/>
     {* 通过 script 插件收集加载组件化 JS 代码 *}
     {%script%}
          require.async("home:static/ui/B/B.js");
     {%endscript%}
  {%endhead%}
  {%body%}
      {%require "home:static/index/index.css"%}
      {%widget "home:widget/A/A.tpl"%}
  {%endbody%}
{%endhtml%}
