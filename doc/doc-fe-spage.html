spage
  • 功能：使用 ajax 请求，采取 quickling 模式渲染页面片段。第一次刷新页面的时候
  • 使用：{%spage "$id" for="pages-container"%}
  • $id: html 片段的路径
  • for：页面容器，即 ajax 请求 html 片段在页面对应的容器。
  • 用法：quickling 模式渲染页面片段
    {%html framework="home:static/lib/mod.js"%}
  {%head%}
     <meta charset="utf-8"/>
  {%endhead%}
  {%body%}
      <div id="pages-container">
      {% spage "spa:widget/pagelets/home/home.tpl" for="pages-container" %}
   </div>
  {%endbody%}
{%endhtml%}
