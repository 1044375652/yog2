调试模式启动

Linux

# project目录
cd yog
npm i #执行过可忽略
npm run-script debug

Windows

# project目录
cd yog
npm i #执行过可忽略
npm run-script debug-win

访问 http://127.0.0.1:8080 即可查看运行效果

注意，如果端口8080被占用会提示启动异常EADDRINUSE，只需要调整bin/www文件中的默认端口即可 也可以设置PORT环境变量来调整启动端口
